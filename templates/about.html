About
Ajmal Adam

[Developer]

Ajmal Adam is a Software Engineer, Ethical Hacker, Blockchain Researcher and Film Maker from Kerala - india. He specialises in Ai Smart Contract Security, vulnerability analysis, and building tools that make on-chain systems safer and more auditable.

His approach combines automated tooling, targeted fuzzing, and manual review to produce practical, verifiable findings and remediation guidance. He contributes to open-source tooling and shares reproducible research to help developers and auditors improve security standards.

Operating between Karuvarakundu (Malappuram) and Kochi, Ajmal focuses on applied research at the intersection of blockchain, Ai-driven auditing, and space-grade communication systems.

Core technologies and tools: Solidity • Python • C++ • Java Script • Remix IDE • Foundry • Slither • Manticore • Mythril • Echidna. and more

Research Areas

Smart contract security • Static analysis • Fuzzing • Symbolic execution • Audit automation • Space-tech integration

Tools & Ecosystems

Vision
